QR CODE

Generate a QR code for the current page or any custom text instantly. Minimal, local, and fast — built with Tailwind for a compact popup.

Features

  • Auto-generate a QR for the active tab when you open the extension
  • Editable input field — change text/URL and the QR updates live
  • Copy QR image to clipboard or download as PNG
  • Generates QR codes locally — no external servers
